# Online-Retail-Sales-Analysis-Database

Overview

This project focuses on building a structured database for an online retail system using MySQL. The goal is to organize customer, product, and order data efficiently and use SQL queries to generate meaningful insights that can help understand sales performance and customer behavior.

Technology Stack
Database: MySQL 8.0
Tool: MySQL Workbench
Language: SQL

Key Features & Analysis

Top Selling Products

Valuable Customer Identification

Monthly Revenue

Category-wise Analysis

Inactive Customer Detection

Getting Started

To run this project on your system, follow these steps:

1. Clone the repository
   git clone https://github.com/Sathwik672/Online-Retail-Sales-Analysis-Database.git

2. Create the database schema
   Open MySQL Workbench and run the schema_design.sql file

3. Insert sample data
   Execute the data_insertion.sql script to populate the tables

4. Run analysis queries
   Execute the insights.sql file to view the outputs
